      女优订阅与更新追踪

      把已抓女优沉淀为长期订阅，后续只检查新增影片并一键回填到 JAV 爬虫。

      支持扫描本地 filmData.json 自动建基线，也支持手动输入演员名称、演员爬取地址、演员番号数新增订阅。

      订阅引导

      • 2. 点击“订阅更新”后，软件会对比上次同步数与当前总数。
      • 3. 发现更新后，直接点“更新订阅”即可只抓新增影片。
