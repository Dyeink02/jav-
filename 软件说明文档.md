# 软件说明文档

## 1. 软件定位

`JAV 自动集成源` 是 Windows 桌面工具，提供三类能力：

1. JAV 爬虫：抓取影片与磁力
2. 视频整理：按番号整理本地视频
3. AV 订阅：检测更新并执行更新抓取

## 2. 运行环境

- Windows 10/11 x64
- 建议使用可用代理
- 保证输出目录有足够磁盘空间

## 3. 功能区说明

### 3.1 JAV 爬虫

可配置：

- 抓取地址
- 抓取数量/总页数/每页条数
- 并行量、请求延迟、超时
- 代理地址
- Cloudflare 兼容开关

输出：

- `filmData.json`
- `magnet-links.txt`
- 运行日志与质量摘要

### 3.2 视频整理

用于本地视频的批量识别、重命名、归类和清理。  
建议先完成爬虫，再执行整理。

### 3.3 AV 订阅

用于对已订阅女优做更新检测与更新抓取。  
当前策略：

1. 先扫描页面番号并与基线对比
2. 得到待更新番号后再启动更新抓取
3. 抓取后仅保留待更新目标结果

## 4. 标准使用流程

1. 在 `JAV 爬虫` 配置参数并开始抓取
2. 检查运行日志与结果文件
3. 需要整理时进入 `视频整理`
4. 需要追更时进入 `AV 订阅`

## 5. 常见问题

### 5.1 代理不通

先确认代理地址和端口可用，再看软件中的代理状态提示。

### 5.2 命中 Cloudflare/年龄检测

开启 Cloudflare 兼容并确认代理可用。  
如果依然失败，优先检查网络环境，再复盘日志。

### 5.3 订阅检测到更新但输出为空

通常是“待更新番号未抓到”或“回收过滤后为空”。  
请查看订阅日志中的“待更新番号”“回收结果”“未抓到番号”条目。

## 6. 输出与日志建议

1. 每次任务尽量使用独立输出目录
2. 保留 `run-时间戳/logs` 目录便于复盘
3. 反馈问题时附上完整日志目录路径
